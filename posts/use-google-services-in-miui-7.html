在 MIUI-7 系统中使用 Google 全家桶

在安卓手机上使用谷歌服务的解决方案。

Why

  • Q:为什么我要使用谷歌的服务?
  • A:主要是国内的厂商太坑了,小米最近也开始 GG,各种广告,各种无意义的 app,还没法删除,甚至没法停用.

PS: 其实最直接的诱因还是因为我在小米的应用商店里面甚至找不到一款可以愉快玩耍的 2048..

How

开始记录折腾的过程.

FuckGFW

手机上的翻墙我还是用的 shadowsocks,不过是 shadowsocks-android,需要到 github 上去获取最新版本的 apk 安装包 ( 因为国内厂商的应用商店里面明显不会有这种东西 ).

基本步骤就是到 github 上 shadowsocks-android 项目的 releases 页面 去下载 ( 当前 ) 最新版本的 apk 包,然后安装到设备上就可以了.

为了偷懒,还是放一个 github 下载地址吧: shadowsocks-nightly-2.9.11.apk ( 因为我决定在不久的将来把手机刷成原生的 android 系统 )

使用 shadowsocks 需要账号一枚,不过这里就不详细记录了,没意思.

然后需要在设备上对 shadowsocks 进行一下配置,我在折腾了许久之后选择的方案是 设备上全局代理 + 流量 绕过局域网和中国大陆地址 1

Get apks for google play services

接下来需要获取谷歌应用服务的一系列 apk 包. 一开始在百度上各种搜索不到,然后开了梯子到 google 上搜索,一下子就搜到了.

最后搜索到这个神奇的网站 apk-mirror, 这里面有各种应用的各种版本的 apk 包, 可以根据自己的机型选择合适的 apk 安装2. 继续偷懒,放下载地址:

  • 首先是 Google Play services 8.7.03 (2645110-030)
  • 然后是 Google Play Store 6.2.13.A-all 0 2655766

Footnotes:

1

一开始为了防止流量浪费,使用的分应用代理,然后发现各种不方便,因为每安装一个应用都需要:

  1. 跑到 ss 里面先把连接断开,
  2. 把新装的应用添加到代理列表里面,
  3. 再启动连接.
2

个人感觉还是一定要来这个网站下载 apk 包,而不是用墙内的什么 谷歌应用安装器 之类的东西, 有几个缺点:

  • 首先是那些软件不知道有多久没有维护了,而且制作也很简陋,因为使用过所以说说,应用安装后在使用的过程中会出现各种奇葩的错误.
  • 然后是安装的软件版本老旧,用那种软件安装的谷歌软件都是很久很久以前的版本了,跟新的系统可能产生兼容性问题.
